==============================
INSTRUCTIONS FOR RUNNING SERVER
===============================

1. In a command line window, navigate to the folder containing this README.txt file
2. Run the command "rmiregistry 1099"
3. Open a new command line window and navigate to the same folder again
4. Run the command "java -jar ChatServer.jar" to start the server on localhost
	To specify a different IP to listen on, append it to the end of the command as an argument
	for example "java -jar ChatServer.jar 192.168.1.94"
